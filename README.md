# SecureVault AES Encryption

## Problem Statement

Sensitive data needs strong encryption to prevent unauthorized access. AES, a symmetric key algorithm, provides robust security and efficiency, making it ideal for protecting files, messages, and databases.

## Tech Stack

- Python
- cryptography
- PyQt5
- Matplotlib

## Installation

1. Clone this repository.
2. Create a virtual environment and activate it.
3. Install the dependencies: `pip install -r requirements.txt`.
4. Run the project: `python main.py`.

## Usage

Run the script with the default settings to see sample output. Modify the code to integrate with real data sources or user interfaces.
